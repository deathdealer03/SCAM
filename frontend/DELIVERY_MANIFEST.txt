SUPPORT & TROUBLESHOOTING
================================================================================

Common Issues:

1. Build Fails
   Solution: rm -rf node_modules && npm install

2. Animations Stutter
   Solution: Reduce particleCount, enable GPU acceleration

3. High CPU Usage
   Solution: Lower particle count, increase animation durations

4. API Errors
   Solution: Check backend running, verify CORS headers

5. Blur Not Working
   Solution: Already has webkit prefix in CSS
